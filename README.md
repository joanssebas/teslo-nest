# teslo db api

1. clonar proyecto

2. `yarn install`
3. clonar el env template y renombrarlo a env
4. cambiar las variables de entorno

5. levantar la base de datos

6. ejecutar seed

```
http://localhost:3000/api/seed
```

7. levantar modo desarrollo `yarn start:dev`

```
docker-compose up -d
```
